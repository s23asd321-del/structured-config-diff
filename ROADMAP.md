# Roadmap

## 0.1.x

- Stabilize the diff model.
- Keep JSON and TOML as the only supported formats.
- Improve examples and report readability.
- Add focused tests for edge cases in nested arrays and max-depth behavior.

## Later

- Consider configurable report sections.
- Consider optional path allowlists.
- Consider more precise redaction keyword configuration.
- Consider package publishing once usage and API boundaries are clear.

## Out Of Scope For The First Version

- YAML support.
- Network access.
- AI-generated advice.
- Web UI.
- Database integration.
- Plugin system.
- Automatic repair or write-back.
- Secret scanning, vulnerability scanning, or compliance judgment.
