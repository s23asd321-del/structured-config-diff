# Contributing

Thank you for considering a contribution.

Please keep changes aligned with the first-version scope:

- Local-first behavior.
- Read-only comparison.
- JSON and TOML support only.
- No network access.
- No telemetry.
- No automatic fixes.
- No unnecessary dependencies.

Before opening a pull request:

- Run `python -m pytest`.
- Add or update tests for behavior changes.
- Update `README.md` or `docs/` when CLI behavior or report format changes.
- Keep security, privacy, and legal wording conservative.
- Do not include real credentials, tokens, logs, private endpoints, personal emails, or production configuration files.
