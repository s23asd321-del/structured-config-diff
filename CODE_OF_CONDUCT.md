# Code Of Conduct

This project expects respectful, constructive collaboration.

Participants should:

- Be clear and kind in issues, reviews, and discussions.
- Focus criticism on code, documentation, tests, and project scope.
- Avoid harassment, personal attacks, and discriminatory language.
- Avoid posting private configuration data, credentials, logs, or other sensitive material.

Maintainers may remove content or decline participation that harms the project or its contributors.
